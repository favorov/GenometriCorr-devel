Genometric Correlation (GenometriCorr): installation procedure

We hope to see our package in Bioconductor eventually, so currently we keep the package in our own installation repository.

  • First, install the IRanges and GenomicRanges, rtracklayer and TxDb.Hsapiens.UCSC.hg19.knownGene.html packages from Bioconductor. 
    source("http://bioconductor.org/biocLite.R")
biocLite("IRanges")
biocLite("GenomicRanges")
biocLite("rtracklayer")
biocLite("TxDb.Hsapiens.UCSC.hg19.knownGene")

  • Install gplots: 
    install.packages("gplots")
    or, 
    source("http://bioconductor.org/biocLite.R")
biocLite("gplots")
  • Install RCurl: 
    install.packages("RCurl")
    or, 
    source("http://bioconductor.org/biocLite.R")
biocLite("RCurl")
    It is critical, intallation process can stuck otherwise.

  • To install the package itself, run this command in R: 
    install.packages('GenometriCorr',repos='http://genometricorr.sourceforge.net/R/',type='source')
  • If you prefer GitHub, the following will do the same: 
    library(devtools)
install_github('favorov/GenometriCorr')
  • Now, all the packages you need are installed in you R system. Each time you start a new R session going to use GenometriCorr , say: 
    library('GenometriCorr')
  • Enjoy!
