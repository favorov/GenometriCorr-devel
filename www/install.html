Genometric Correlation (GenometriCorr): installation procedure

We hope to see our package in Bioconductor eventually, so currently we keep the package in our own installation repository.

  • First, install Bioconductor 
    if (!requireNamespace("BiocManager"))
    install.packages("BiocManager")
BiocManager::install()
    Then, use Bioconductor's BiocManager to install rtracklayer and TxDb.Hsapiens.UCSC.hg19.knownGene.html packages from Bioconductor.. 
    if( ! require('rtracklayer') ) BiocManager::install("rtracklayer")
if( ! require('TxDb.Hsapiens.UCSC.hg19.knownGene') ) BiocManager::install("TxDb.Hsapiens.UCSC.hg19.knownGene")

  • Install gplots: 
    if(!require('gplots')) install.packages("gplots")
    or, 
    if(!require('gplots')) BiocManager::install("gplots")
  • Install RCurl: 
    if(!require('RCurl')) install.packages("RCurl")
    or, 
    if(!require('RCurl')) BiocManager::install("RCurl")
    It is critical, intallation process can stuck otherwise. Usually, the most of the packages are already installed.

  • To install the GenometriCorr package itself, run this command in R: 
    install.packages('GenometriCorr',repos='http://genometricorr.sourceforge.net/R/',type='source')
  • If you prefer GitHub+devtools, the following will do the same: 
    library(devtools)
install_github('favorov/GenometriCorr')
  • Now, all the packages you need are installed in you R system. Each time you start a new R session going to use GenometriCorr , say: 
    library('GenometriCorr')
  • Enjoy!
